### Overview
A Windows desktop tool for batch-converting PNG, JPEG, and HEIC images into EPS format, commonly required for academic paper submissions.  
This tool can streamline manual EPS conversion.

### Features
- Batch conversion from PNG / JPEG / HEIC to EPS
- Supports folders and multiple files
- Drag-and-drop user interface
- Preview of converted images
- Fully offline operation

### Technology Stack
- Python
- Pillow
- pillow-heif
- Tkinter / tkinterdnd2
- PyInstaller (installer creation)
